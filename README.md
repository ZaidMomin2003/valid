# Cleanmails - Email Verification & List Management

Cleanmails is a comprehensive tool for email validation, list cleaning, and deliverability optimization.

## Features

- **Email Validation**: Instant verification of email lists with risk analysis.
- **Email Extractor**: Extract email addresses from raw, unstructured text.
- **List Cleaner**: Clean and normalize messy CSV/Excel files.
- **Spam Analysis**: Deep analysis of email content to identify spam triggers.
- **Full Setup Service**: Dedicated infrastructure solutions for high-volume outreach.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
